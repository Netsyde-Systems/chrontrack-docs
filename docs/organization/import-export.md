# Import & Export

You can export your entire task tree as a JSON file and import it back later, or import a tree from another Chrontrack account.

## Exporting your task tree

1. Go to **Organize** (`/organize`).
2. Scroll to the **Export** section at the bottom of the task tree panel.
3. Click **Export JSON**. The file downloads immediately.

The export includes all task nodes with their hierarchy, names, colors, and icons. Time entries and tags are not included.

## Importing a task tree

1. Go to **Organize** (`/organize`).
2. Click **Import JSON** and select a previously exported file.
3. The imported tree is merged into your existing hierarchy as new top-level nodes.

> Importing does not overwrite your existing tree — it appends to it.
