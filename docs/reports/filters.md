# Reports & Filters

The **Reports** page (`/reports`) lets you explore your time data with flexible filters and a collapsible breakdown tree.

![Reports page](/img/screenshots/reports.png)

## Period filter

Choose a preset period or set a custom date range:

| Option | Description |
|---|---|
| Today | Current calendar day |
| This week | Monday–Sunday of the current week |
| This month | Current calendar month |
| All time | No date constraint |
| Custom | Pick a start and end date |

## Category filter

Click the **Category** selector to restrict the report to a specific task node and all its descendants. Leave it unset to include all tasks.

## Team / member filter

> This filter is only visible if you are a **Collaborator** in at least one team.

1. Choose a **team** from the dropdown.
2. Choose a **member** (or "All members") to see their entries alongside yours.

## Tag filter

Click the **Tags** field and select one or more tags. Entries must carry at least one of the selected tags to be included (OR logic).

## Breakdown tree

The breakdown section shows your tasks as a collapsible tree. Each node displays the total time logged under it (including all descendants). Expand a node to see how time was distributed across its children.

## Entry list

Below the breakdown, the full list of matching time entries is shown in chronological order with task path, tags, and duration.

## Total badge

The header shows the **total duration** for all entries matching the current filters.
