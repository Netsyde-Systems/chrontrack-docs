# Export

You can export the currently filtered report as a **PDF** or **CSV** file.

## Exporting as PDF

1. Apply any filters you want on the [Reports](filters.md) page.
2. Click the **Export PDF** button.
3. A formatted PDF is generated and downloaded immediately.

The PDF includes the breakdown tree and the full entry list for the selected filters.

## Exporting as CSV

1. Apply your filters.
2. Click **Export CSV**.
3. A CSV file is downloaded with one row per time entry.

### CSV columns

| Column | Description |
|---|---|
| Date | Entry date |
| Task | Full task path (e.g. `Professional › Work`) |
| Tags | Comma-separated tag names |
| Start | Start time |
| End | End time |
| Duration | Duration in `HH:MM:SS` format |

## Tips

- Use the **period** and **category** filters to narrow your export to a billing period or a specific project before downloading.
- The **team/member** filter lets you export a single team member's time for payroll or invoicing purposes.
