# Navigation

Chrontrack's interface is organized around a top navigation bar with links to each major feature area.

![Navigation bar](/img/screenshots/nav.png)

## Navigation links

| Icon | Label | Path | Purpose |
|---|---|---|---|
| Clock | **Timer** | `/` | Start and stop time tracking |
| Bar chart | **Reports** | `/reports` | Filter and export time data |
| Circles | **Goals** | `/goals` | Manage time targets |
| People | **Teams** | `/teams` | Collaborate with others |
| Person | **Dependants** | `/managed-accounts` | Manage tracked accounts |
| Checklist | **Organize** | `/organize` | Edit task hierarchy and tags |

## User menu

The user menu in the top-right corner provides:

- **Profile** — edit your name and password
- **Switch account** — jump into a managed (dependant) session
- **Sign out**

## Managed sessions

When you're acting as a managed account, a banner at the top of the screen shows the account name and a **Return** button to switch back to your own session.
