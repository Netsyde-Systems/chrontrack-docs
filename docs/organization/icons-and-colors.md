# Icons and Colors

Each task node can have a custom icon and color, making it easy to scan your task tree at a glance.

![Icon and color pickers](/img/screenshots/icons-colors.png)

## Setting a color

Click the color swatch next to a task name to open the color picker. Choose from the preset palette or enter a hex value. The color is applied to the task's icon and used as an accent throughout the app (timer picker, reports breakdown).

## Setting an icon

Click the icon button next to a task name to open the icon picker. Browse or search the available icons — these come from the [Lucide](https://lucide.dev) icon set.

## Inheritance

Colors and icons are per-node only — they do not inherit from parent to child automatically. Set each node individually for the look you want.
